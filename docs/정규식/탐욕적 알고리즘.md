## 탐욕적 알고리즘
**탐욕적 수량자**  
최대한으로 매칭을 시도한다.

\* : 0번이상 반복되면 매칭된다.  
\+ : 1번이상 반복되면 매칭된다.  
{n} : 정확히 n개만을 찾습니다.  
{n,} : n개 이상을 찾습니다.  
{n,m} : 최소 n개, 최대 m개의 경우를 찾습니다.


**게으른 수량자**  
최소한으로 매칭을 시도한다.

*? : 0번이상 반복되면 매칭된다.  
+? : 1번이상 반복되면 매칭된다.  
{n,}? : n개 이상을 찾습니다.
