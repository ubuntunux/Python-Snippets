## 모든 파일 & 폴더 출력하기

```
    import os
    workDIr = os.path.abspath('.')
    for dirpath, dirnames, filenames in os.walk(workDIr):
        print dirpath
    
        for dirname in dirnames:
            print "\t", dirname
            
        for filename in filenames:
            print "\t", filename
```
