## 절대경로 , 상대경로 , 현재경로
python 에서 경로처리에 관한 모듈은 os.path 를 사용한다.

현재 실행되는 스크립트파일의 절대경로를 구하려면

    os.path.dirname( os.path.abspath( __file__ ) )

두 경로의 상대적 경로 구하기 

    os.path.relpath( "c:\\ik" , "c:\\os\\pp" )
    '..\\..\\ik'

공통경로 구하기 

    os.path.commonprefix( ["c:\\ik\\jj" , "c:\\ik\\sd" ])
    'c:\\ik\\'

중간 경로 슬래쉬 제거 

    os.path.normpath( "c:\\os\\..\\pp" )
    'c:\\pp'
  
현재 실행경로 구하기 

    os.getcwd()

경로명 합치기 

    os.path.join( "a" , "b" ,"c" )
    'a/b/c'
